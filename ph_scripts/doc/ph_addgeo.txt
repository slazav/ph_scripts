ph_addgeo -- добавление географических координат в exif-данные фотографий.

usage: [<options>] <jpeg files>
options:
  -t --track    -- plt track file;
  -s --shift    -- time shift (hours), default 0;
  -f --datafile -- datafile, default .geodata.txt;
  -v --verbose  -- be more verbose.
  -D --maxdist  -- max distance between interpolated points, m (default: 200)
  -T --maxtime  -- max time between interpolated points, s (devault: none)

Координаты и высота фотографий берутся из трека (в формате
OziExplorer), если он задан опцией -t. Важно, чтобы часы на
фотоаппарате были синхронизированы с gps (если это не так, время можно
сдвинуть опцией -s). Данные трека линейно интерполируются, при этом
можно установить ограничения на максимальный диапазон интерполяции по
времени (-M) и по расстоянию (-D).

Кроме того, данные берутся из файла .geodata.txt. Данные из файла
имеют приоритет. Это нужно, чтобы можно было исправить неправильные
координаты у отдельных фотографий.

Координаты записываются в exif-данные фотографий, кроме того,
записывается обновленный файл .geodata.txt.


    Пример

- берем данные из трека track.plt со сдвигом на 5 часов:
$ ph_addgeo -t 5 -v -t track.plt *.jpg
- данные из трека попали в фотографии и в .geodata.txt
- теперь можно подредактировать .geodata.txt: добавить недостающие
  координаты, удалить неправильные...
- обновляем exif-данные:
$ ph_addgeo *.jpg
